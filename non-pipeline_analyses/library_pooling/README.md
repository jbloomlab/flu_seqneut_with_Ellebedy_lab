# Library pooling

This directory describes the initial pooling of assessing viral barcode balancing, and assessing the fraction of viral barcode reads on increasing plated MDCK-SIAT1 cells. This library took the 78 H3 strains in Kikawa et al (2025), and 5 additional circulating H3's, as well as additional H1's from [Link Text] https://github.com/jbloomlab/flu_seqneut_pdmH1N1_2023-2024.  Authors: Ariana Ghez Farrell and  Caroline Kikawa

## Input

All data for the notebooks are pulled directly from seqneut-pipeline output.
