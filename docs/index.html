Sequencing-based neutralization assays of human serum samples from the Ellebedy lab with individuals vaccinated either with the 2022-2023 Fluarix egg-based vaccine or with the 2022-2023 mRNA-1010 vaccine, using a ~100 variant library of H3N2 and H1N1 influenza variants.

The numerical data and computer code are at https://github.com/jbloomlab/flu_seqneut_with_Ellebedy_lab
